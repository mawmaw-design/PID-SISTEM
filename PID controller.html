LOGIKA KONTROL "HYBRID"

Inovasi kami menggabungkan kecepatan dan presisi:

üöÄ MODE 1: TURBO

Kondisi: Suhu < 36.5¬∞C

Memaksa PWM 100% (255). Mengabaikan PID untuk mengejar waktu pemanasan secepat mungkin.

üéØ MODE 2: PID STABIL

Kondisi: Suhu 36.5¬∞C - 37.5¬∞C

Mengaktifkan perhitungan PID halus. Menjaga suhu agar mendarat lembut di 37.0¬∞C tanpa lonjakan.

LOGIKA KODE: SAFETY & TIMER

Fitur koding penting untuk responsivitas dan keamanan sistem:

// 1. NON-BLOCKING TIMER (MILLIS)
unsigned long currentMillis = millis(); if (currentMillis - prevMillis > 200) { prevMillis = currentMillis; runSystem(); }

Menggunakan millis() agar Arduino tidak berhenti bekerja. Jika menggunakan delay(), pembacaan sensor akan "buta" selama jeda waktu tersebut (blocking).

// 2. SAFETY CUT-OFF LOGIC
if (inputSuhu > 37.5) { pidOutput = 0; // PAKSA MATI integral = 0; // RESET MEMORI PID digitalWrite(pinLedRed, HIGH); tone(pinBuzzer, 1000); }

Baris kode ini adalah Hard Limit. Memaksa output pemanas menjadi 0 segera setelah suhu melewati batas, mengabaikan perhitungan PID demi keselamatan bayi.
